# CPE 歷屆考題分析：難度 × 主題 × 真實答對率

> judge. — 內部數據報告 · v4
> 資料來源：cpe.mcu.edu.tw 歷屆試題公開測資（2013–2026，51 屆，334 題）· 主題分類為人工／AI 判讀 · 即時數據另見平台後台 `/admin/analytics`

分析 2013–2026 年間 **51 屆**有公開測資的 CPE 考試，涵蓋全部 A～G 七個題目位置（共 334 道不重複題目），並補上官網公布的逐屆平均答對題數，以及**逐題的真實應試考生答對／答錯人數**。

## 重點發現

難度曲線非常清楚，而且這次有真實考生數據驗證：**A 題真實答對率 80.7%**，一路單調遞減到 **G 題只剩 31.4%**。主題上同樣吻合——**A、B 題幾乎只考數學／字串／陣列**，DFS/BFS/圖論與動態規劃是 **0 題**；從 **D 題開始 DP 明顯增加**，**E 題起圖論大量出現**，到 **F、G 題，圖論與 DP 合計超過一半**。真實答對率的單調遞減，直接驗證了「用 A～G 位置代替官方星等」這個假設是合理的。

---

## 方法論與重要提醒

**關於「星等」的定義**：CPE 官網沒有在考題列表頁公布每題的官方星等，我們資料庫裡的難度欄位對所有 CPE 題目都是預設值「1」，**並非真正的官方分級**。本報告改用「**題目在該屆的位置（A～G）**」作為難度的替代指標——這是根據 CPE 一貫由易到難排序的慣例所做的合理推斷，仍是一項**假設**，並非官方認證。

主題分類（334 題全部）由人工／AI 逐題判讀考題敘述得出，分類本身涉及主觀判斷；少數幾題（PDF 測資已從官網下架、僅能靠題名推測）在下方表格中會特別標註。

**資料格式提醒**：CPE 官網從 2025-09-30 這屆開始換了新的網址格式，且不再直接公布預先算好的期望輸出，改成公布「參考解答原始碼」——本報告的資料是**下載參考解答、自行編譯執行**來產生期望輸出，而非直接採用官網現成答案。
